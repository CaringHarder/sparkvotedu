  ```typescript
  const pollActivities = await Promise.all(
    polls.map(async (poll) => {
      let hasVoted = false
      if (pid) {
        const voteCount = await prisma.pollVote.count({
          where: { pollId: poll.id, participantId: pid },
        })
        hasVoted = voteCount > 0
      }
      return {
        id: poll.id,
        name: poll.question,
        type: 'poll' as const,
        participantCount: poll._count.votes,
        hasVoted,
        status: poll.status,
      }
    })
  )
  ```

**Create src/app/api/polls/[pollId]/upload-url/route.ts** -- Signed upload URL endpoint:
- POST endpoint
- Requires teacher auth: getAuthenticatedTeacher()
- Validates input: z.object({fileName: z.string().min(1), contentType: z.string().regex(/^image\/(jpeg|png|webp|gif)$/)})
- Generates path: `poll-options/${teacherId}/${Date.now()}-${fileName}`
- Uses Supabase admin client to create signed upload URL: supabase.storage.from('poll-images').createSignedUploadUrl(path)
- Returns JSON: {signedUrl, token, path} or {error}
- NOTE: The 'poll-images' bucket must be created in Supabase Storage. Add a comment noting this requires manual bucket creation.
