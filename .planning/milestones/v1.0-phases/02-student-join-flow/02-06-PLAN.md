<task type="auto">
  <name>Task 2: Replace dashboard shell placeholder with session quick-actions</name>
  <files>
    src/components/dashboard/shell.tsx

  <action>
Update `src/components/dashboard/shell.tsx` to replace the static placeholder with actionable session content:

1. Add imports:
   - `Link` from 'next/link'
   - `getTeacherSessions` from '@/lib/dal/class-session'
   - `Plus` and `ArrowRight` icons from 'lucide-react'

2. After the existing `getAuthenticatedTeacher()` call, fetch recent sessions:
   ```ts
   const sessions = await getTeacherSessions(teacher.id)
   const activeSessions = sessions.filter(s => s.status === 'active')
   ```

3. Replace the placeholder div (the `<div className="rounded-lg border bg-card p-8 text-center text-muted-foreground">` block containing "Your brackets and polls will appear here.") with TWO sections:

   **Section A - Quick Actions row:**
   ```tsx
   <div className="grid gap-4 sm:grid-cols-2">
     <Link
       href="/sessions"
       className="flex items-center gap-3 rounded-lg border bg-card p-4 transition-colors hover:bg-accent"
     >
       <div className="flex h-10 w-10 items-center justify-center rounded-md bg-primary text-primary-foreground">
         <Plus className="h-5 w-5" />
       </div>
       <div>
         <p className="font-medium">Create Session</p>
         <p className="text-sm text-muted-foreground">Start a new class session</p>
       </div>
     </Link>
   </div>
   ```

   **Section B - Active Sessions summary (only if activeSessions.length > 0):**
   ```tsx
   {activeSessions.length > 0 && (
     <div className="space-y-3">
       <div className="flex items-center justify-between">
         <h2 className="text-lg font-semibold">Active Sessions</h2>
         <Link
           href="/sessions"
           className="flex items-center gap-1 text-sm text-muted-foreground hover:text-foreground"
         >
           View all <ArrowRight className="h-4 w-4" />
         </Link>
       </div>
       <div className="grid gap-3 sm:grid-cols-2 lg:grid-cols-3">
         {activeSessions.slice(0, 3).map((session) => (
           <Link
             key={session.id}
             href={`/sessions/${session.id}`}
             className="rounded-lg border bg-card p-4 transition-colors hover:bg-accent"
           >
             <p className="font-medium">{session.name || 'Unnamed Session'}</p>
             <p className="mt-1 font-mono text-lg font-bold">{session.code}</p>
             <p className="mt-1 text-sm text-muted-foreground">
               {session._count.participants} student{session._count.participants !== 1 ? 's' : ''}
             </p>
           </Link>
         ))}
       </div>
     </div>
   )}
   ```

   If there are NO active sessions, show an encouraging empty state instead of the old placeholder:
   ```tsx
   {activeSessions.length === 0 && (
     <div className="rounded-lg border bg-card p-8 text-center">
       <p className="text-muted-foreground">No active sessions yet.</p>
       <p className="mt-1 text-sm text-muted-foreground">
         Create a session to get your students engaged.
       </p>
     </div>
   )}
   ```

4. Keep the welcome heading and tier badge unchanged at the top.

IMPORTANT: DashboardShell is an async Server Component. The `getTeacherSessions` call is a direct Prisma query -- this is the correct pattern (Server Component fetches data directly, no API call needed). This matches the established pattern from sessions/page.tsx.
