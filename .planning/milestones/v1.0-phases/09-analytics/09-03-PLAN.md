  <files>src/actions/analytics.ts</files>
  <action>
Create `src/actions/analytics.ts` with 'use server' directive. All actions follow existing patterns from `src/actions/bracket.ts` and `src/actions/poll.ts`.

**getBracketExportData(bracketId: string)**
- Auth: `getAuthenticatedTeacher()` -> return `{ error: 'Not authenticated' }` if null
- Tier check: `canAccess(teacher.subscriptionTier as SubscriptionTier, 'csvExport')` -> return `{ error: 'CSV export requires Pro plan', upgradeTarget }` if not allowed
- Ownership: `getBracketWithDetails(bracketId, teacher.id)` -> return `{ error: 'Bracket not found' }` if null
- Fetch `getBracketVoteDistribution(bracketId)` from analytics.ts
- Transform to flat export rows: `Array<{ Round: number, Position: number, 'Entrant 1': string, 'Entrant 2': string, Winner: string, 'Entrant 1 Votes': number, 'Entrant 2 Votes': number, 'Total Votes': number }>`
- Sort by round asc, position asc
- Return `{ data: rows, filename: bracketName }` where filename is the bracket name sanitized for filesystem (replace non-alphanumeric with underscores)

**getPollExportData(pollId: string)**
- Auth + tier check (same pattern as above)
- Ownership: `getPollByIdDAL(pollId)` -> verify teacherId matches
- Fetch `getPollVoteDistribution(pollId, poll.pollType)` from analytics.ts
- Transform to flat export rows: `Array<{ Option: string, Votes: number, Percentage: string }>` for simple polls
- For ranked polls: `Array<{ Option: string, 'Borda Score': number, Percentage: string }>`
- Return `{ data: rows, filename: pollQuestion }`

**getPredictiveExportData(bracketId: string)**
- Auth + tier check: check `teacher.subscriptionTier === 'pro_plus'` (predictive brackets are Pro Plus only)
- Ownership: verify bracket exists and is owned by teacher, and bracketType is 'predictive'
- Fetch `scoreBracketPredictions(bracketId)` from prediction.ts
- Calculate totalRounds from bracket size (same Math.ceil(Math.log2(effectiveSize)) pattern)
- Transform to flat rows: `Array<{ Rank: number, 'Student Name': string, 'Total Points': number, 'Correct Picks': number, 'Total Picks': number, 'Accuracy': string, ...roundColumns }>` where roundColumns are `'R1 Points': number, 'R2 Points': number, ...` for each round from 1 to totalRounds
- Import `getPointsForRound` from `@/lib/bracket/predictive` to show point values in column headers: `'R{n} ({pts}pts)'`
- Sort by totalPoints desc (already sorted by scoring engine)
- Return `{ data: rows, filename: bracketName + '_predictions' }`

All three actions: use try/catch with generic error return `{ error: 'Failed to export data' }` on unexpected errors.
