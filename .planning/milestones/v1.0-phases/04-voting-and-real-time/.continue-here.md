2. **"Vote Submitted!" confirmation card** - Added motion/react AnimatePresence with three-state card transitions (matchup -> confirmation -> next matchup) in simple voting mode. Shows "Vote Submitted!" with checkmark for 1200ms before sliding to next card. File: simple-voting-view.tsx

3. **"Back to brackets" navigation** - Added Link from next/link with left arrow SVG on student bracket pages (draft, completed, simple, advanced views). File: src/app/(student)/session/[sessionId]/bracket/[bracketId]/page.tsx

4. **Student extra animation vs teacher** - Student WinnerReveal onComplete triggered celebration immediately, teacher didn't. Removed the extra trigger. Files: simple-voting-view.tsx, advanced-voting-view.tsx

5. **"Active Activities" heading** - Changed to "Brackets/Polls". File: activity-grid.tsx

6. **Cramped activity cards** - Redesigned from horizontal to vertical/stacked layout with line-clamp-2 for names, larger icons. File: activity-card.tsx

7. **QR code + join code on teacher live dashboard** - Rewrote QRCodeDisplay as compact chip button with expandable dropdown showing QR code, large code text, copy buttons. Added sessionCode prop to LiveDashboard, fetched session code in live page server component. Files: qr-code-display.tsx, live-dashboard.tsx, live/page.tsx

8. **Bracket delete cascade failure** - Vote model's `entrant` FK to BracketEntrant was missing `onDelete: Cascade`. Added it and pushed schema via `prisma db push`. File: prisma/schema.prisma

9. **Student stuck on last matchup card** - `safeIndex` was clamped to `votableMatchups.length - 1`, making "all voted" check impossible. Fixed by using raw `currentIndex >= votableMatchups.length` and added "All votes in!" card inside AnimatePresence. File: simple-voting-view.tsx

10. **Leftover WinnerReveal after dismissing CelebrationScreen** - CelebrationScreen appears at 4s while WinnerReveal still plays (~7.5s). Dismissing celebration revealed WinnerReveal underneath. Fixed by having onDismiss also clear revealState in all three views. Files: simple-voting-view.tsx, advanced-voting-view.tsx, live-dashboard.tsx

11. **"Advance 1 matchup" on completed bracket** - Button showed for final round. Added `currentRound < totalRounds` condition and "Bracket Complete!" state with trophy icon. File: round-advancement-controls.tsx
