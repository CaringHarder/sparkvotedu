    **2. Create src/components/bracket/winner-reveal.tsx** (`'use client'`):

    Dramatic countdown + winner announcement overlay. Used when teacher "reveals" a matchup result.

    Props: `{ winnerName: string; entrant1Name: string; entrant2Name: string; onComplete: () => void; showVoteCounts?: boolean; entrant1Votes?: number; entrant2Votes?: number }`

    Animation sequence (using `motion` from `"motion/react"`):
    1. Full-screen overlay fades in (semi-transparent dark background)
    2. Countdown: 3... 2... 1... (each number scales up from 0.5 to 1.5 with spring animation, holds briefly, then exits)
    3. "And the winner is..." text fades in with slight upward movement
    4. Brief suspenseful pause (500ms)
    5. Winner name appears with a scale-up + glow animation (large bold text, centered)
    6. If `showVoteCounts` is true: vote count display slides in below winner name (e.g., "7 votes to 3")
    7. After 3 seconds, calls `onComplete` callback to dismiss
    8. Optional: user can click/tap anywhere to dismiss early

    Implementation:
    - Use `motion.div` with `animate` prop for each stage
    - Use `AnimatePresence` for enter/exit transitions
    - Stagger timing with `transition: { delay: X }` on each element
    - Dark overlay: `bg-black/70 fixed inset-0 z-50`
    - Countdown numbers: use `motion.span` with `scale` and `opacity` keyframes
    - Winner name: large (text-5xl md:text-7xl), bold, primary color
    - Support `prefers-reduced-motion`: skip animations and show result directly (use `motion`'s `useReducedMotion` hook or CSS media query)

    **3. Create src/components/bracket/celebration-screen.tsx** (`'use client'`):

    Full celebration screen for the championship winner. Shown when the final bracket matchup is decided.

    Props: `{ championName: string; bracketName: string; onDismiss: () => void }`

    Structure:
    - Full-screen overlay (z-50)
    - Champion announcement:
      - "CHAMPION!" large text with scale animation
      - Champion name in huge bold text with glow/shadow effect
      - Bracket name in smaller text below
      - Trophy or crown emoji/icon (simple, large)
    - Confetti:
      - Call `confetti()` from canvas-confetti on mount
      - Multiple bursts: one from left, one from right, one from center
      - Use `confetti({ particleCount: 100, spread: 70, origin: { y: 0.6 } })` for center burst
      - Add staggered bursts: 0ms, 300ms, 600ms for a cascade effect
      - Pass `disableForReducedMotion: true` for accessibility
    - "Continue" button at bottom to dismiss
    - Auto-dismiss after 10 seconds if not manually dismissed
    - Motion animations for text elements (fade in, scale up)

    **Important notes:**
    - Import motion as: `import { motion, AnimatePresence } from "motion/react"` (NOT from "framer-motion")
    - Import confetti as: `import confetti from 'canvas-confetti'`
    - Both components must respect `prefers-reduced-motion`
    - Both are overlays (portal or fixed position) that don't disrupt layout
