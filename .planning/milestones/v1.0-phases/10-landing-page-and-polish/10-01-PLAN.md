         ```
         --brand-amber-dark: oklch(0.65 0.17 75);
         ```
       - Add dark-mode overrides in the `.dark` block (brand blue stays similar, amber adjusts for contrast):
         ```
         --brand-blue: oklch(0.72 0.13 230);
         --brand-blue-light: oklch(0.25 0.05 230);
         --brand-blue-dark: oklch(0.60 0.15 230);
         --brand-amber: oklch(0.78 0.15 75);
         --brand-amber-light: oklch(0.25 0.06 75);
         --brand-amber-dark: oklch(0.68 0.17 75);
         ```
       - Register brand colors in the `@theme inline` block:
         ```
         --color-brand-blue: var(--brand-blue);
         --color-brand-blue-light: var(--brand-blue-light);
         --color-brand-blue-dark: var(--brand-blue-dark);
         --color-brand-amber: var(--brand-amber);
         --color-brand-amber-light: var(--brand-amber-light);
         --color-brand-amber-dark: var(--brand-amber-dark);
         ```
       - Update the dark variant from `@custom-variant dark (&:is(.dark *));` to `@custom-variant dark (&:where(.dark, .dark *));` for proper Tailwind v4 compatibility (targets .dark element itself, zero specificity).

       - Three options: Light, Dark, System
       - Sun icon visible in light mode, Moon visible in dark mode (CSS rotation transition)
       - Export as named `ThemeToggle`
