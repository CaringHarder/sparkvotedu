    **Test 1: Single Elimination (8 entrants)**

    5. Toggle to "Visual Placement" mode
    6. Verify: Pool shows all 8 entrants with seed badges on the left

    10. Click an entrant in pool, then click an empty slot -- click-to-place should work
    11. Click "Auto Seed" -- all entrants placed in default order

    5. Verify bye count stays at 3 (never increases or decreases)

    1. Create bracket with 32 entrants (use topic list for quick fill)
    2. Switch to Visual Placement

    1. Create Round Robin bracket, size 6

    **Test 5: Edit existing draft bracket**
    1. Create a draft bracket (don't activate)

    3. Verify: Visual Placement toggle is available
    4. Switch to visual mode, make changes, save
    5. Return to detail page -- seeding should reflect changes

    **Test 6: Mobile responsive**
    1. Open browser DevTools, toggle mobile view (375px width)
    2. Create a bracket with Visual Placement
    3. Verify: Pool appears above the matchup grid (not sidebar)
    4. Click-to-place works on mobile
