## Problem

Round robin brackets in simple vote mode require the user to press "Next" to navigate between matchups and the matchup display looks small/cramped. In contrast, single elimination brackets in simple mode present matchups in a much more engaging, full-sized way. The round robin experience feels inferior and inconsistent.

## Solution

- Audit the single elimination simple vote mode UI for how it presents matchups to students
- Update round robin simple vote to use the same full-sized matchup presentation (no manual Prev/Next navigation, auto-advance or streamlined flow)
- Match the card sizing, layout, and interaction pattern from single bracket simple mode
