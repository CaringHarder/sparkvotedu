**Create src/hooks/use-poll-vote.ts** -- Optimistic vote hook:
- 'use client'
- Props: pollId, participantId, pollType ('simple' | 'ranked'), allowVoteChange, existingVotes (for restoration)
- State: selectedOptionId (simple) or rankings ({optionId, rank}[]) (ranked), submitting boolean, submitted boolean, error string|null
- `selectOption(optionId)` -- For simple: sets selectedOptionId. For ranked: should not be used (separate hook or different method).
- `submitVote()` -- Calls castPollVote server action. For simple: sends {pollId, participantId, optionId}. Sets submitting=true, then submitted=true on success.
- `canChangeVote` -- true if allowVoteChange and poll is still active
- Initializes from existingVotes prop: if existingVotes has entries, pre-select the option (simple) or rankings (ranked) and set submitted=true.
- Returns: {selectedOptionId, submitting, submitted, error, selectOption, submitVote, canChangeVote}

**Create src/components/student/ranked-poll-vote.tsx** -- Tap-to-rank:
- 'use client' component
- Props: poll (PollWithOptions), participantId, existingVotes
- State: rankings array of {optionId, rank}, max rankings = poll.rankingDepth ?? poll.options.length
- Interaction model: Tap-to-rank. First tap on an option = #1 (shows "1" badge), second tap on a different option = #2, etc.
- Tapping an already-ranked option: does nothing (use undo/reset instead)
- Number badges: circular badge with rank number in top-left of card. Colors: #1=gold, #2=silver, #3=bronze, rest=muted
- "Undo Last" button: removes the last-added ranking. Visible when rankings.length > 0.
- "Reset All" button: clears all rankings. Visible when rankings.length > 0.
- Ranking counter: "Ranked X of Y" indicator (Y = rankingDepth or total options)
- Submit button: "Submit Rankings" -- enabled when rankings.length === maxRankings (user must rank the required number). If rankingDepth is set, must rank exactly that many. If null (rank all), must rank all options.
- Calls castPollVote server action with {pollId, participantId, rankings}
- After submission: show "Rankings submitted!" success state
- If canChangeVote: show "Change Rankings" button
- Restore previous rankings from existingVotes prop (map to rankings array, show as pre-selected with badges)
- Touch targets: same card style as simple vote but with rank badge overlay
- Cards layout: vertical stack (1 column) on mobile for easier tap-to-rank ordering, 2 columns on tablet+
