**Emoji regex:** Use a comprehensive Unicode range regex that covers:
- Emoticons: U+1F600-U+1F64F
- Misc Symbols & Pictographs: U+1F300-U+1F5FF

- Flags: U+1F1E0-U+1F1FF
- Misc: U+2600-U+26FF, U+2700-U+27BF
- Variation selectors: U+FE00-U+FE0F
- Supplemental: U+1F900-U+1F9FF, U+1FA00-U+1FA6F, U+1FA70-U+1FAFF

**Step 1: Create the UpgradeBanner component**

Create `src/components/dashboard/upgrade-banner.tsx` as a client component.

**Implementation requirements (from user locked decisions):**
- Banner copy: "We've upgraded! Previous sessions have been cleared."
- Dismissible: has a dismiss/close button
- Shown once: after dismissal, never shows again
- First teacher login after migration: shows on dashboard load

**Dismissal mechanism (Claude's discretion per research):** Use localStorage with key `sparkvotedu_upgrade_banner_dismissed`. On mount, check if key exists. If absent, show banner. On dismiss, set key to current ISO timestamp. If localStorage is unavailable (private browsing, SSR), fail silently -- do not show the banner (better to miss the notification than show it forever).

**Component design:**
- Use `'use client'` directive (localStorage access requires client component)
- Use `useState` for visibility, `useEffect` for localStorage check on mount
- Style with Tailwind: blue-50 background, blue border, blue text. Match the existing dashboard aesthetic. Use rounded corners, subtle shadow.
- Include a dismiss button (X icon or "Dismiss" text). Use the existing X icon from lucide-react if available, otherwise a text button.
- Responsive: works on mobile and desktop
- Animated: fade out on dismiss (simple opacity transition, nothing complex)

**Component structure:**
```tsx
'use client'
import { useState, useEffect } from 'react'
import { X } from 'lucide-react'  // Check if lucide-react is installed; if not, use text "Dismiss"

const BANNER_KEY = 'sparkvotedu_upgrade_banner_dismissed'

export function UpgradeBanner() {
  const [visible, setVisible] = useState(false)

  useEffect(() => {
    try {
      if (!localStorage.getItem(BANNER_KEY)) {
        setVisible(true)
      }
    } catch {
      // localStorage unavailable -- don't show
    }
  }, [])

  function dismiss() {
    setVisible(false)
    try {
      localStorage.setItem(BANNER_KEY, new Date().toISOString())
    } catch {
      // Silently fail
    }
  }

  if (!visible) return null

  // Blue info banner matching SparkVotEDU brand
  return (
    <div className="...">
      <p>We&apos;ve upgraded! Previous sessions have been cleared.</p>
      <button onClick={dismiss} aria-label="Dismiss notification">
        <X className="h-4 w-4" />
      </button>
    </div>
  )
}
```

Use proper Tailwind classes: `bg-blue-50 dark:bg-blue-950/30 border border-blue-200 dark:border-blue-800 rounded-lg p-4 mb-4 flex items-center justify-between gap-3`. Support dark mode since the project has a ThemeToggle component.
