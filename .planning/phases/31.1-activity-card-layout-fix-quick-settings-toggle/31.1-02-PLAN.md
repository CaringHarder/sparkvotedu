    **Step 3: Create server action** in `src/actions/bracket.ts`:
    ```tsx
    export async function updateBracketViewingMode(input: unknown) {
      // 1. Authenticate teacher
      const teacher = await getAuthenticatedTeacher()

      // 2. Validate input with Zod
      const { bracketId, viewingMode } = updateBracketViewingModeSchema.parse(input)

      // 3. Ownership check -- verify bracket belongs to this teacher
      //    Use the existing pattern from other bracket actions (fetch bracket, check teacherId)
      const bracket = await prisma.bracket.findUnique({ where: { id: bracketId }, select: { teacherId: true } })
      if (!bracket || bracket.teacherId !== teacher.id) throw new Error('Bracket not found')

      // 4. Update viewingMode in database
      await prisma.bracket.update({
        where: { id: bracketId },
        data: { viewingMode },
      })

      // 5. Broadcast settings_changed event
      await broadcastBracketUpdate(bracketId, 'settings_changed', { viewingMode })

      // 6. Revalidate paths
      revalidatePath('/activities')
      revalidatePath(`/brackets/${bracketId}`)

      // 7. Return success
      return { success: true }
    }
    ```

    Follow the exact patterns from existing server actions in the file:
    - Import `getAuthenticatedTeacher` from the auth utility (check existing imports in bracket.ts)
    - Import `prisma` from the database utility
    - Import `broadcastBracketUpdate` from broadcast.ts
    - Import the Zod schema from validation.ts
    - Use `revalidatePath` from `next/cache`
    - Mark as `'use server'` at top of file if not already present

    **Decision (Claude's discretion):** This action works on all bracket states (draft, active, paused, completed) per locked decision. No status check needed -- viewing mode is a display-only setting that doesn't affect bracket integrity.

    **Decision (Claude's discretion):** No auto-adjustment of other settings when switching modes. If switching simple->advanced, seeds/vote counts/timer settings remain unchanged.
