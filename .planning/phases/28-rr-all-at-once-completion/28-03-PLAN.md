<objective>
Fix the "Close All & Decide by Votes" button so it works for every round in all-at-once RR brackets, not just the highest round number.

Purpose: In all-at-once mode, `currentRoundRobinRound` always equals the highest active round (e.g., 3). The batch decide handler hardcodes its filter to that value, so clicking the button on rounds 1 or 2 produces an empty matchup array and silently does nothing. This is a blocker found during UAT.

Output: Two-file fix -- prop type change + handler parameter change -- enabling per-round batch decide.
</objective>
