  <action>
Create two new files:

**src/lib/poll/csv-parser.ts** -- Poll option CSV parser following the bracket csv-parser.ts pattern exactly:
- Export `ParsedPollOption` interface: `{ text: string; imageUrl?: string }`
- Export `parsePollOptionCSV(file: File): Promise<ParsedPollOption[]>`
- Use papaparse with `header: true`, `skipEmptyLines: true`, `transformHeader: (h) => h.trim().toLowerCase()`
- Text column aliases: `row['text'] || row['option'] || row['name'] || Object.values(row)[0]` -- trim result
- Image column aliases: `row['image'] || row['imageurl'] || row['photo'] || row['url']` -- trim result, set to `undefined` if empty
- Filter out entries where text is empty after trimming
- Reject on papaparse error

**src/lib/utils/csv-image-upload.ts** -- Shared utility for downloading external image URLs and re-uploading to Supabase Storage:
- Export `ImageUploadResult` interface: `{ publicUrl: string | null; error?: string }`
- Export `downloadAndReuploadImage(externalUrl: string, uploadEndpoint: string): Promise<ImageUploadResult>`
  - fetch the external URL, check response.ok
  - Get blob, validate blob.type starts with 'image/'
  - Determine contentType (use blob.type if jpeg/png/webp, default to 'image/jpeg')
  - Generate fileName: `csv-import-${Date.now()}-${Math.random().toString(36).slice(2,8)}.${ext}`
  - POST to uploadEndpoint with { fileName, contentType } to get { signedUrl, publicUrl }
  - PUT blob to signedUrl with Content-Type header
  - Return { publicUrl } on success
  - Catch all errors, return { publicUrl: null, error: message } -- never throw
- Export `processCSVImages(items, uploadEndpoint, onProgress, onWarning): Promise<Map<string, string>>`
  - Takes array of `{ id: string; pendingImageUrl?: string }` items
  - Filters to items with pendingImageUrl
  - Processes sequentially (not parallel -- avoids rate limits per research pitfall 5)
  - Calls onProgress(completed, total) after each item
  - Calls onWarning(id, error) on failure
  - Returns Map of id -> publicUrl for successful uploads
