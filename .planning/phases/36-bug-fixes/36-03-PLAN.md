**Step 1: Rewrite name-disambiguation.tsx**

Replace the current candidate list UI with a simpler inline re-prompt:

1. The component should show a clear message: "Name taken. Add your last initial to join."
2. The tone is short and direct -- not playful (per user decision)
3. Show an input field pre-filled with the original name the student typed (e.g., "David")
4. The student should be able to edit the input to add their last initial (e.g., "David R.")
5. A submit button to try the modified name
6. If the student submits the exact same name again: block and re-prompt with the same message ("Name taken. Add your last initial to join."). The server already returns duplicates again via `joinSessionByName` when an exact match exists.
7. Remove the candidate list rendering and "That's me!" / "I'm someone different" flow for the initial join experience
