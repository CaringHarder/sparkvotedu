<feature>
  <name>Round-Robin Engine</name>
  <files>src/lib/bracket/round-robin.ts, src/lib/bracket/__tests__/round-robin.test.ts</files>
  <behavior>
  Pure functions. No DB calls.

  generateRoundRobinRounds(entrantCount: number) -> RoundRobinRound[]:
  Uses the circle method (fix seed 1, rotate others).

  For 4 entrants: 3 rounds, 2 matchups per round, 6 total matchups
    Round 1: (1v4, 2v3)
    Round 2: (1v3, 4v2)
    Round 3: (1v2, 3v4)

  For 5 entrants (odd): 5 rounds, 2 matchups per round (one bye per round), 10 total matchups
    Each entrant plays 4 matches and sits out 1 round

  For 8 entrants: 7 rounds, 4 matchups per round, 28 total matchups

  Validation: throws if entrantCount < 3 or > 8

  Key properties to test:
  - Total matchups = N*(N-1)/2
  - Each entrant appears exactly once per round (for even N) or at most once (for odd N)
  - No duplicate pairings across all rounds
  - Every possible pairing appears exactly once

  calculateRoundRobinStandings(results: RoundRobinResult[]) -> RoundRobinStanding[]:
  Input: Array of { entrant1Id, entrant2Id, winnerId: string | null }
    - winnerId = entrant1Id or entrant2Id -> that entrant wins
    - winnerId = null -> tie

  Scoring: Win = 3 points, Tie = 1 point, Loss = 0 points

  Ranking: Sort by points descending, then wins descending.
  Head-to-head tiebreaker: If two entrants have equal points AND equal wins, the one who beat the other in their direct matchup ranks higher.

  Output: RoundRobinStanding[] sorted by rank (1-indexed rank field).
  </behavior>
  <implementation>
  1. generateRoundRobinRounds:
     - If odd: add phantom entrant (entrantCount+1 as BYE placeholder)
     - Fix entrant 1, rotate [2, 3, ..., N] using circle method
     - Each round: pair i with N-1-i from the arranged list
     - Skip pairings involving the BYE placeholder
     - Return RoundRobinRound[] with roundNumber (1-indexed) and matchups array

  2. calculateRoundRobinStandings:
     - Accumulate W/L/T and points for each entrant
     - Sort by points desc, wins desc
     - For ties: check head-to-head result
     - Assign rank (1-indexed, equal rank for unresolvable ties)
  </implementation>
</feature>
