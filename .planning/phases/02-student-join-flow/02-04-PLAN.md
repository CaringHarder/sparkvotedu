- Add `'use client'` directive

- Import `joinSession` from `@/actions/student`
- Import `useRouter` from `next/navigation`
- Import `useState` from 'react'
- UI: Single large text input for 6-digit code, styled prominently as the hero element
  - Input type="text", inputMode="numeric", pattern="[0-9]*", maxLength={6}, placeholder="Enter class code"

- Error UX: shake animation on invalid code, red border, clear error message
