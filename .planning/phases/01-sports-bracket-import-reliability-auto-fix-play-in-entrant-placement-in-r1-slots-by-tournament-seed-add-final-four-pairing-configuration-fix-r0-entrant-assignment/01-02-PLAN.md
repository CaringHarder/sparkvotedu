   - For each R0 matchup where the game is closed and has a winnerId:
     a. Determine the winning team from the SportsGame data (compare homeTeam.externalId and awayTeam.externalId to game.winnerId).

     d. Log: `console.log('[play-in-resolve]', winner.abbreviation, 'wins play-in, replacing combined entry')`.
   - This resolution runs on every sync, but the update is idempotent (if already resolved, the entrant already has externalTeamId set, so the `IS NULL` filter skips it).
