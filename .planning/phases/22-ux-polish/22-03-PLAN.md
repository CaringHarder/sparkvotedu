<objective>
Unify activation terminology to "Start" / "Active" / "End" across all brackets and polls, and update session dropdowns to show session names instead of just codes.

Purpose: Teachers find "Activate" vs "Go Live" confusing -- this is a classroom tool, not a streaming platform. Simpler language ("Start" / "End") is more intuitive for action buttons. Navigation links to view an already-active session use "View Live" (distinct from "Start" which is the action that activates). Session dropdowns showing names (e.g., "Period 3 History (ABC123)") instead of just codes helps teachers quickly identify which session to link activities to.

Output: 9 terminology changes across 6 files, 4 session dropdown updates across 3 files, dashboard shell badge and name fallback updates, 2 page server component data fixes
</objective>

The locked decision "Unified to Start (NOT Go Live)" specifically targets the activation action. Navigation links to view a live activity are a separate concern and use "View Live" for clarity.
