<objective>
Remove all FingerprintJS code, dependencies, and database columns from the codebase.

Purpose: FingerprintJS was the original browser identity mechanism, now fully replaced by localStorage persistence (Phase 42) and name-based identity (Phases 40-41). Removing it eliminates ~150KB from the client bundle and cleans up dead code across the application, schema, and package layers.

Output: Clean codebase with zero fingerprint references, smaller bundle, updated privacy page, and audit-trail migration SQL.
</objective>
