    Polish the placement UI interactions to match all locked decisions:

    **Bye slot handling:**
    - Bye slots render 'BYE' in italic muted style: `className="italic text-muted-foreground"` with a muted background
    - Byes ARE draggable (per locked decision: teachers can move byes to different slots)
    - When a bye is dragged to an occupied slot, the entrant and bye swap positions
    - Bye slots are droppable targets (entrants can be placed in a bye slot, pushing the bye elsewhere)
    - Bye count is fixed: `bracketSize - entrantCount`. The system never creates or destroys byes.
    - Byes are identified by slot index: if `slotIndex` is in `getByeSlots(entrantCount, bracketSize)` AND no entrant has been manually placed there, it's a bye

    **Remove from slot:**
    - Placed entrants in slots show a small X button (lucide X icon, `h-3.5 w-3.5`)
    - Clicking X "removes" the entrant from visual placement by resetting their seedPosition
    - "Removing" means: swap this entrant with whatever is in their original seed position (effectively undoing the manual placement for this one entrant)
    - Alternatively, simpler approach: the X button sends the entrant back to the auto-seed position for their original seed. Use `autoSeed` to get the default positions, then only reset this one entrant.
    - Actually, simplest: clicking X on a slot swaps this entrant back to their auto-seed position. Call `placeEntrantInSlot(entrants, entrantId, autoSeedSlot, bracketSize)`.

    **Drag back to pool:**
    - The EntrantPool area itself is a droppable zone (useDroppable with `data: { type: 'pool' }`)
    - Dragging a placed entrant back to the pool area triggers the same "remove from slot" behavior

    **Auto-seed button:**
    - At the top of the EntrantPool, show an "Auto Seed" button (lucide Wand2 or similar icon)
    - Clicking it calls `autoSeed(entrants)` and `onEntrantsChange(result)`
    - This resets all entrants to their natural list order (seed 1 in slot 1, etc.)
    - Add a confirm if entrants are already manually placed: "Reset all placements to default seeding?"

    **Real-time matchup update:**
    - The placement bracket grid shows matchup pairings that update immediately as entrants are placed
    - Each matchup card shows both entrant names (or BYE) for its two slots
    - When an entrant is placed in a slot, the matchup card immediately reflects the new pairing

    **Selected entrant visual:**
    - When an entrant is selected (click-to-place mode), show ring-2 ring-primary on the pool item
    - All valid empty/bye slots pulse with a subtle animation when an entrant is selected (same as drag glow)
    - Clicking another entrant in the pool deselects the first and selects the new one
    - Clicking the same entrant again deselects it
    - Pressing Escape deselects
